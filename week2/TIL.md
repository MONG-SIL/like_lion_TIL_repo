### 2. 핵심 정리 (내 언어로)
•	Media Query (반응형)
화면 크기(px)에 따라 다른 레이아웃을 적용하는 방식이다.
모바일에서는 한 줄로 배치하고, 화면이 커질수록 2열, 3열로 바뀌도록 설정했다.

•	CSS 적용 우선순위 (Specificity)
같은 요소에 여러 스타일이 적용될 때 어떤 스타일이 적용될지를 결정하는 기준이다.
!important > inline style > id > class > tag 순서로 우선순위가 높다.
실제로 스타일이 적용되지 않을 때 우선순위를 확인하는 것이 중요하다.

•	position 속성
요소를 원하는 위치에 배치할 때 사용하는 속성이다.
relative를 기준으로 absolute를 사용해 특정 요소를 카드 위에 배치하는 식으로 활용했다.
