### 2. 핵심 정리 (내 언어로)
전체적인 구조를 생각하면서 디자인을 생각하자
- flexbox : 요소들을 가로/세로 방향으로 정렬하는 레이아웃 방식 → 카드 내부 요소 정렬에 사용

- grid : 2차원(행/열) 기반 레이아웃 시스템 → 상세 정보 영역을 2열로 구성

- linear-gradient, radial-gradient : 색상을 부드럽게 이어주는 그라데이션 같은 배경 효과
