# 동료 피드백 기록

배포 URL: https://week10-ochre.vercel.app

## 제공한 피드백 (내가 동료에게)

### 익명 동료 1

- **잘된 점:** Supabase에 저장돼서 새로고침해도 명단이 그대로 남아 있음
- **개선할 점:** 모바일에서 요약 카드가 좁아서 한 줄 소개가 잘림
- **추가 제안:** 상세 페이지에서 이전·다음 사람으로 넘어가는 버튼이 있으면 좋겠음

### 익명 동료 2

- **잘된 점:** 비로그인 상태에서도 명단은 볼 수 있고, 수정만 로그인 필요해서 역할이 분명함
- **개선할 점:** 요약 카드가 링크인지 한눈에 안 들어옴
- **추가 제안:** 필터나 검색을 쓰면 몇 명이 걸렸는지 숫자로 보여주면 편할 것 같음

## 받은 피드백 (동료가 나에게)

### 익명 동료 1

- **잘된 점:** 파트 필터, 정렬, 이름 검색이 한 화면에서 잘 맞물려 동작함
- **개선할 점:** 모바일에서 랜덤 추가 버튼 줄이 답답해 보임
- **추가 제안:** 로그인 안내 배너가 눈에 잘 들어와서 좋음 (유지 요청)

### 익명 동료 2

- **잘된 점:** 회원가입·로그인 실패 시 한국어 메시지가 이해하기 쉬움
- **개선할 점:** 카드에 마우스를 올려도 클릭해서 상세로 갈 수 있다는 느낌이 약함
- **추가 제안:** 검색·필터 적용 후 "몇 명 나왔는지" 표시해 주면 목록이 비었을 때 덜 헷갈림

### 익명 동료 3

- **잘된 점:** 배포 URL로 폰에서도 접속해서 확인할 수 있었음
- **개선할 점:** 상세 페이지 URL을 붙여넣기 했을 때 처음엔 404가 났음 (`/#/` 형태로 들어가야 함)
- **추가 제안:** _(없음)_

## 반영한 개선 사항

| 피드백 | 반영 내용 | 관련 파일 |
|--------|-----------|-----------|
| 카드 클릭 가능성이 잘 안 보임 (익명 동료 2) | 카드 hover·`focus-visible` 강조, 카드 하단에 **상세 보기** 문구 추가 | `src/styles/style.css`, `src/components/LionSummaryCard.tsx` |
| 모바일 레이아웃 (익명 동료 1) | 640px 이하에서 헤더·fetch 영역·카드 세로 배치 | `src/styles/style.css` |
| 검색·필터 결과 개수 표시 (익명 동료 2) | 필터/검색으로 목록이 줄면 **전체 N명 중 M명 표시** 안내 | `src/components/SummarySection.tsx`, `src/pages/HomePage.tsx` |
